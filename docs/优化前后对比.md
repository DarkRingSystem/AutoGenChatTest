## 问题 2：首页缺少矩阵雨背景

### 修改前 ❌

```
层级结构：
┌─────────────────────────────────────┐
│  ModeSelector                       │
│  ├─ background: 纯色渐变            │
│  ├─ Logo                            │
│  ├─ 标题                            │
│  └─ 卡片                            │
└─────────────────────────────────────┘

背景效果：
- 深色主题：紫色渐变
- 浅色主题：蓝灰渐变
- 无矩阵雨
- 无动态效果
```

**问题**：
- ❌ 没有矩阵雨背景
- ❌ 没有动态球体
- ❌ 与聊天界面风格不一致
- ❌ 视觉效果单调

---

### 修改后 ✅

```
层级结构：
┌─────────────────────────────────────┐
│  mode-selector-wrapper              │
│  ├─ background: 主题色              │
│  ├─ MatrixRain (矩阵雨)             │
│  ├─ animated-background             │
│  │  ├─ gradient-orb-1              │
│  │  ├─ gradient-orb-2              │
│  │  └─ gradient-orb-3              │
│  └─ ModeSelector                    │
│     ├─ background: transparent      │
│     ├─ Logo                         │
│     ├─ 标题                         │
│     └─ 卡片 (半透明毛玻璃)         │
└─────────────────────────────────────┘

背景效果：
- 深色主题：黑色背景 + 矩阵雨 + 动态球体
- 浅色主题：浅色背景 + 矩阵雨 + 动态球体
- 与聊天界面风格一致
- 视觉效果丰富
```

**优势**：
- ✅ 有矩阵雨背景
- ✅ 有动态球体
- ✅ 与聊天界面风格一致
- ✅ 视觉效果丰富
